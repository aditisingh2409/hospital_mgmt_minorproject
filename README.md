# 🏥 Hospital Patient Management System

## 📌 Overview
The Hospital Patient Management System is a Python-based mini project designed to manage patient information in hospitals and clinics. It allows users to register patients, assign doctors, generate bills, and display patient records through a simple menu-driven interface.

## ✨ Features
- Patient Registration
- Doctor Assignment
- Bill Generation
- Display Patient Records
- Menu-Driven Program

## 🛠️ Technologies Used
- Python 3
- VS Code
